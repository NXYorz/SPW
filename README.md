# SPW 学习计划打卡（云端部署版）

一个面向计算机学生的 **学习计划打卡 + 学习资料库** 网站。支持多用户注册、云端同步，并具备“公开动态”功能，方便同学间互相监督激励。

- **UI**：React + TailwindCSS + TDesign (`tdesign-react@1.12.0`)
- **后端**：Node.js (Express) + PostgreSQL
- **架构**：Docker Compose (Nginx + API + DB)
- **多端同步**：数据保存在云端数据库，登录后自动同步
- **社交属性**：支持将计划设为“公开”，全站用户可在动态中看到

## 功能概览

- **账号体系**
  - 用户注册/登录：默认普通用户，拥有私有计划空间
  - **管理员账号**：内置账号 `admin`，负责维护公共学习资料库

- **计划打卡 (云端同步)**
  - 月视图日历：展示每日进度；点击日期管理详细计划
  - **公开/私有**：计划可设为公开。公开后，其他同学在同一天的“全站动态”里能看到你的动态
  - 实时同步：支持开启“自动刷新”，实时获取云端更新

- **学习资料库**
  - 分类检索：C++、算法、操作系统、网络、Node.js 等
  - 管理员维护：管理员可随时增删改查资料，全站实时同步

## 部署与运行

项目已通过 Docker Compose 部署在 Lighthouse 服务器上。

### 1）访问地址
`http://140.143.131.225/` (或你绑定的域名)

### 2）管理员信息
- **用户名**：`admin`
- **初始密码**：`spw-admin`
- *建议登录后在“管理员”页修改密码*

### 3）本地开发环境 (可选)
如果你想在本地继续修改代码：
1. `cd study-checkin`
2. `docker compose up -d` (需要本地安装 Docker)
3. 访问 `http://localhost`

## 技术细节

- **Nginx**：作为入口，处理静态文件并反向代理 `/api` 流量到后端。
- **PostgreSQL**：持久化存储用户信息、学习计划、资料数据。
- **JWT**：使用 JSON Web Token 进行身份验证，确保存储在云端的数据安全性。
